## Сборка `.docx`

```powershell
pwsh ./build.ps1
```

Скрипт:
1. Конкатенирует пронумерованные `.md`-файлы в порядке `00_… → 08_…` в `99_full.md`.
2. Через Word COM (`Word.Application`) создаёт `.docx` со стилями по ГОСТ 2.105:
   - Шрифт основного текста: Times New Roman, 14 pt;
   - Межстрочный интервал: 1.5;
   - Отступ первой строки абзаца: 1.25 см;
   - Поля: левое 30 мм, правое 10 мм, верхнее/нижнее 20 мм;
   - Выравнивание основного текста по ширине.
3. Сохраняет `build/VKR_Villa_Jaconda.docx`.

Требования: Windows, установленный Microsoft Word (любая версия с COM-поддержкой).
